## [1.0.0] - 2026-01-14

### Added
- Initial repository structure
- Comprehensive README.md with extensive cultural sensitivity warnings
- api.json following Wyrdness schema v1.0.0 with cultural respect metadata
- SOURCES.md bibliography prioritizing Navajo and Indigenous authors
- Issue templates for reports and corrections (with cultural sensitivity guidelines)
- Pull request template

### Data Included
- Phenomenon metadata and classification (SHAPESHIFTER/Indigenous Tradition)
- Cultural sensitivity warnings and respect guidelines
- Traditional Navajo understanding of Yee naaldlooshii
- Geographic distribution (Navajo Nation, Southwestern USA)
- Sacred/taboo nature documentation
- Appropriation concerns section
- Theories emphasizing Navajo spiritual belief system
- Navajo resources and consultation guidelines
- Commercial exploitation clarification (Skinwalker Ranch)

### Cultural Sensitivity Features
- Critical-level cultural warnings throughout
- Sacred knowledge designation
- DO/DO NOT guidelines for respectful engagement
- Prioritization of Navajo voices
- Appropriation harm documentation
- Resources for proper cultural consultation
